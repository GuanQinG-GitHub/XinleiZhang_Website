1. Make sure you have installed CasADi.
	if not, please install the toolbox via https://web.casadi.org/get/

2. Make sure you have installed the Peter Corke's Robotic Toolbox for MATLAB.
	if not, please install the toolbox via https://petercorke.com/toolboxes/robotics-toolbox/

3. Then, run the MATLAB script named ARM_3DOF_MultipleShooting_3D.m for the direct optimization method;
	run the script named ARM_3DOF_MultipleShooting_3D_mpc.m for the MPC method.

4. Run the script Comparison_Plot.m will produce result plots.
